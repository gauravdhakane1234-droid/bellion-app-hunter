# 🔍 Bellion App Hunter

Mass search, analyze and rank money-making apps from Google Play Store and web.

## What it does

- Searches Play Store for earning apps
- Verifies payment proofs from Reddit/Twitter
- Ranks apps by real reliability score
- Database of 12+ apps with daily payouts

## Quick Start

```bash
python3 scripts/play_store_scraper.py search "earn money"
python3 scripts/play_store_scraper.py rank
python3 scripts/payment_proof_finder.py Frizza
```
